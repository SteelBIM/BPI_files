New in Origin 7.0

Many new features have been added to Origin 7.0 to increase both ease-of-use and power. These include In-place Text Editing, Expanded Drawing Tools, a Graph Template Library, a Nonlinear Least Squares Fitting Wizard, Categorical Data Support and Drag and Drop Import of ASCII and Thermo Galactic SPC files. We have improved our statistical tests and added Two-Way ANOVA and Survival Analysis. Now you can automate your analysis and plotting routines using the all-new Origin C programming language featuring the Code Builder integrated development environment and a library of advanced computational routines from the Numerical Algorithms Group (NAG), Inc.
