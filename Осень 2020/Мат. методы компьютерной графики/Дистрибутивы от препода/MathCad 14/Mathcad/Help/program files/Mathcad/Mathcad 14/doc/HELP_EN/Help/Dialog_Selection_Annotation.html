Selection Annotation Dialog Box

The Selection Annotation dialog allows you to note and track the sourcing of expressions in a Mathcad worksheet.

Comment tab

The Comment tab allows you to note source information for all or part of an expression in your worksheet. As you copy and paste the expression into another worksheet, any comment associated with the region is copied along with it.

When annotating a subexpression, such as 1/2·b·h, not all of the selected combinations can be annotated. If Annotate Selection is unavailable, place parentheses to group the subexpression, such as 1/2·(b·h), and then annotate the selection.

Provenance tab

The Provenance tab is visible for an expression once you have copied it to a new worksheet. On this tab, you can see the original source for the expression and the information on the worksheet from which it was most recently copied.
