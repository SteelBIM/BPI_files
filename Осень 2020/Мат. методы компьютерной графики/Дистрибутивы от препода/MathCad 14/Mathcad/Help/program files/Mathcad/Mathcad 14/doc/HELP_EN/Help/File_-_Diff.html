Comparing Two Worksheets

The Compare feature allows you to visually compare the contents of two saved Mathcad worksheets. The two worksheets display side-by-side in the File Comparison window, starting at the first difference. Differences between the two worksheets are outlined with colored rectangles, indicating the type of difference:

  • A red outline indicates content present in only the first (left) worksheet.
  • A green outline indicates content present in only the second (right) worksheet.
  • A yellow outline indicates a region present in both worksheets, but whose content differs between the two.

The current difference is indicated in each worksheet with a thicker border. Click Next and Previous at the top of the screen or use your mouse to move between differences.

You can use the scroll bars to quickly scan the differences in both worksheets. Both sets of scroll bars are synchronized, to maintain a valid side-by-side comparison whenever possible. The display synchronizes at the vertical midpoint of the windows.

The File Comparison window defaults to a magnification of 75%. You can zoom the display from 1% to 100% by using the zoom slider. To view a region at regular size, regardless of the zoom setting, position the mouse over the region.

Viewing the Differences Between Two Worksheets

To view the differences between two worksheets:

  1. Click File > Compare.

  2. Select the two worksheets to compare by clicking Browse to independently select each worksheet. Mathcad initially points you to the current working directory, but you can browse to any folder and file. You can also type the path and filename.

    Comparison only works on saved worksheets. If the selected worksheet has been modified since it was last saved, Mathcad displays a warning that it will perform the comparison on the last saved version. To compare an unsaved worksheet, dismiss the Compare dialog and save the worksheet, then choose File > Compare again.

  3. Click Compare to view the differences between the two files.
  4. When you finish your comparison, click Cancel to dismiss the File Comparison window.

Each comparison only occurs between two worksheets. However, Mathcad allows you to simultaneously view multiple comparisons, including multiple comparisons of the same worksheet.

Notes:

  • Mathcad can compare only XMCD and XMCDZ worksheets. MCD format worksheets cannot be compared.
  • Comparisons of mathematical results are performed using saved results, which are maintained to the full 17 digits of precision, rather than the displayed result. Comparing saved results may indicate differences that are not readily apparent due to the rounding of results for display in Mathcad.
  • Mathcad indicates differences at the region level only. If a difference occurs within a region - for example, a "2" was changed to a "3" in an equation - the entire region is highlighted as different, rather than the individual portion of the region.
